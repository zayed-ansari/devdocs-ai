# DevDocs AI - AI-Powered Codebase Assistant

**AI for Bharat Hackathon 2026**

## Problem
Developers spend 40%+ of their time reading documentation and understanding complex codebases. Manual code exploration is slow, error-prone, and frustrating.

## Solution
DevDocs AI is an AI-powered documentation assistant that helps developers understand codebases through conversational queries and context-aware explanations with source code citations.

## Key Features
- **Semantic Code Search**: Query codebases using natural language
- **Context-Aware Explanations**: RAG-based retrieval for accurate responses
- **Source Citations**: Every answer includes file names, line numbers, and code references
- **Multi-Language Support**: Python, JavaScript, Java, and more
- **Fast Onboarding**: Reduce codebase learning time by 60%

## Tech Stack
- **Backend**: Python, LangChain, FastAPI
- **Vector DB**: ChromaDB
- **LLM**: Gemini API
- **Frontend**: Streamlit
- **Deployment**: Docker

- Improved productivity and debugging efficiency
